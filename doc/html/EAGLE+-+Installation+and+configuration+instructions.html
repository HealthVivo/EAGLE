Software development : EAGLE - Installation and configuration instructions

This page last changed on Feb 18, 2013 by ljanin.

EAGLE @ Illumina

There are 3 EAGLE nests at Illumina:

  • the release versions, which can be found in /illumina/development/EAGLE/EAGLE-<version>
  • the snapshots, which can be found in /illumina/development/EAGLE/EAGLE-snapshot-<date>
  • the development versions, which are on a git server

For the release versions and snapshots, you just need to update your PATH, and you may define a few variables for your convenience:

EAGLE_PATH=/illumina/development/EAGLE/EAGLE-1.5.0
EAGLE_SHARE=$EAGLE_PATH/share/EAGLE
EAGLE_LIBEXEC=$EAGLE_PATH/libexec/EAGLE
PATH=$EAGLE_PATH/bin:$PATH

For the development versions (you may need to setup your git environment as explained here):

# Fetch source code from server
git clone ssh://ukch-git.illumina.com/software/EAGLE.git

# Switch to development branch
cd EAGLE
git branch --track develop origin/develop
git checkout develop
cd ..

# Build
mkdir build
cd build
../EAGLE/src/configure --prefix=<install location>
make install

# Set your environment
EAGLE_PATH=<install location>
EAGLE_SHARE=$EAGLE_PATH/share/EAGLE
EAGLE_LIBEXEC=$EAGLE_PATH/libexec/EAGLE
PATH=$EAGLE_PATH/bin:$PATH

Cluster

If you plan to run EAGLE distributed on Illumina's cluster, make sure you locate either qsub or psubmake (Illumina tool - One version of it is known to be in /illumina/development/LOx/resources/) and add it to your path.

EAGLE on the Amazon Cloud

If you wish to run EAGLE on the Amazon Cloud, we have some RPM ready.

EAGLE at other locations

We can install on demand or provide an RPM or a tar.gz file.
