Preamble


Based on our previous EAGLE Use Case - Refining our Human sequencing simulation (diploid version), we now throw in all our options to make it 2x250 and as realistic as we can:

If you haven't done it yet, you may refer to the EAGLE - Installation and configuration instructions.

Initialisation

The new parameters and configuration files for 2x250 are:

  • --run-info=$EAGLE_SHARE/RunInfo/RunInfo_PairedReads2x251Cycles2x64Tiles.xml

  • --motif-quality-drop-table=$EAGLE_SHARE/MotifQualityDropTables/DefaultMotifQualityDropTable.tsv = realistic quality scores around motifs such as homopolymers

  • --template-length-table=$EAGLE_SHARE/TemplateLengthTables/TemplateLengthTableFrom2x250Run.tsv

  • --quality-table=/home/ljanin/install/EAGLE-release/share/EAGLE/QualityTables/NewQualityTable.read1.length251.qtable2 --quality-table=/home/ljanin/install/EAGLE-release/share/EAGLE/QualityTables/NewQualityTable.read2.length251.qtable2     = Better quality table extracted from real 2x250 runs

source /illumina/development/EAGLE/stable-setup

configureEAGLE.pl \
  --run-info=$EAGLE_SHARE/RunInfo/RunInfo_PairedReads2x251Cycles2x64Tiles.xml  \
  --reference-genome=/illumina/scratch/iGenomes/Homo_sapiens/UCSC/hg19/Sequence/Chromosomes/ \
  --variant-list=$EAGLE_SHARE/Variants/None.vcf \
  --coverage-depth=30 \
  --genome-mutator-options="--organism-ploidy=2 --ploidy-chromosome=chrX --ploidy-level=1 --ploidy-chromosome=chrY --ploidy-level=1 --ploidy-chromosome=chrM --ploidy-level=100" \
  --motif-quality-drop-table=$EAGLE_SHARE/MotifQualityDropTables/DefaultMotifQualityDropTable.tsv \
  --template-length-table=$EAGLE_SHARE/TemplateLengthTables/TemplateLengthTableFrom2x250Run.tsv \
  --quality-table=/home/ljanin/install/EAGLE-release/share/EAGLE/QualityTables/NewQualityTable.read1.length251.qtable2 \
  --quality-table=/home/ljanin/install/EAGLE-release/share/EAGLE/QualityTables/NewQualityTable.read2.length251.qtable2

cd EAGLE

Execution

Same as in the previous tutorial (look back for more ways to run EAGLE):

Running on local machine with 8 processors

make -j 8

This should take about 1 hour to complete.

Running on cluster

psubmake -j 48

or

ssh ukch-dev-lndt01 # recommended (see below)
cd [same directory as before]
make -j 48 sge

This should take about X minutes to complete.
