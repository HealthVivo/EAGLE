RunInfo

Files located in $EAGLE_SHARE/RunInfo:

File

Description

RunInfo_PairedReads1x1Tiles.xml

Configuration file for generation of paired reads in 1 lane x 1 tile

RunInfo_PairedReadsBarcode1x1Tiles.xml

Configuration file for generation of paired reads + barcode in 1 lane x 1 tile

RunInfo_PairedReadsBarcode8x32Tiles.xml

Configuration file for generation of paired reads + barcode in 8 lanes x 32 tiles per lane

RunInfo_SingleRead1x1Tiles.xml

Configuration file for generation of single reads in 1 lane x 1 tile

RunInfo_SingleRead2x2Tiles.xml

Configuration file for generation of single reads in 2 lanes x 2 tiles per lane

Quality Tables

Files located in $EAGLE_SHARE/QualityTables:

File

Description

DefaultQualityTable.read1.length101.qval

Empirical table obtained from the analysis of real Basecalls

DefaultQualityTable.read2.length101.qval

Read 2 qualities, to be used with read 1 table

DefaultQualityTable.barcode.length7.qval

Barcode qualities, currently all set to Q40

ZeroErrorQ50QualityTable.qval

(removed in latest version) Q50 for all bases, which is the special value for zero sequencing errors

Description

Contains a short insertion, a SNP, short deletion and large deletion. Expect a chromosome called "chr" longer than 10Kbases
